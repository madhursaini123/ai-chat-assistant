# AI Chat Assistant 🤖

This project is a conversational AI assistant built using DeepSeek API.

## 🚀 Features
- Chat with AI
- Maintains conversation memory
- Clear chat history
- Exit command

## 🛠️ Tech Used
- Python
- DeepSeek API (LLM)

## ▶️ How to Run

```bash
pip install openai
python main.py
